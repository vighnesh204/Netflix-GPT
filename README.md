# 🎬 Netflix Clone (React Practice Project)

A frontend-focused **Netflix-inspired web application** built using React.
This project is created purely for **learning and practice purposes** to strengthen core React concepts, UI development, and component-based architecture.

---

## 🚀 Overview

This project replicates the basic UI and interaction patterns of a modern streaming platform like Netflix. It is part of my hands-on learning journey and is **inspired by the GitHub repositories of students from the Namaste React course**.

The goal is not to build a full production app, but to **understand how scalable frontend systems are designed** using React.

---

## 🧠 Learning Objectives

* Build reusable and modular **React components**
* Understand **state management** and data flow
* Practice **props handling** and dynamic rendering
* Work with **conditional UI rendering**
* Improve **UI/UX design sense**
* Learn project structuring for real-world applications

---

## 🛠️ Tech Stack

* ⚛️ React (Functional Components + Hooks)
* 🎨 Tailwind CSS (for styling)
* 📦 JavaScript (ES6+)
* ⚡ Vite

## ✨ Features

* 🎥 Netflix-style UI layout
